Tags Estruturais

1- <b> - Essa tag deixa o texto em Negrito.
Exemplo do uso da tag: <p> Esta frase esta em <b> Negrito </b> </p>

2- <i> - Essa tag deixa o texto em Italico.
Exemplo do uso da tag: <p> Esta frase esta em <i> Italico </i> </p>

3- <u> - Essa tag sublinha o texto.
Exemplo do uso da tag: <p> Esta frase esta <u> Sublinhada </u> </p>

4- <br> - Essa tag quebra a linha.
Exemplo do uso da tag: <p> Esta frase esta <br>
Quebrada </p>

5- <span> - Essa tag é usada para aplicar estilo em um trexo especifico, como alterar a cor de uma unica palavra
Exemplo do uso da tag: <p> Esta frase vai ter uma palavra com a <span style = "color: vermelho;" > cor Vermelha </span> </p>

6- <hr> - Essa tag cria uma linha Horizontal para separar as seções.
Exemplo do uso da tag: <p> Após esta frase uma linha Horizontal vai separar as Tags Estruturais das Tags Semanticas <hr> </p>

Tags Semânticas

1- <strong> - Essa tag auxilia os leitores dando Ênfase com semântica de importancia.
Exemplo do uso da tag: <p> Podemos modificar essa frase <strong> dando ênfase nessa parte para auxiliar na leitura </strong> </p>

2- <em> - Essa tag auxilia os leitores dando Ênfase leve como uma semântica de entonação.
Exemplo do uso da tag:<p> Podemos modificar essa frase <em> dando ênfase nessa parte para auxiliar na leitura </em> </p>

3- <blockquot> - Essa tag indica uma citação longa como um paragrafo.
Exemplo do uso da tag:<p> Essa frase pode ter uma citação longa como: <blockquot>

“Um ser humano deve transformar informação em inteligência ou conhecimento. Tendemos a esquecer que nenhum computador jamais fará uma nova pergunta.” Grace Hopper,
</blockquot> </p>

4- <q> - Essa tag indica uma citação curta.
Exemplo do uso da tag:<p> <q> A educação é a arma mais poderosa que você pode usar para mudar o mundo. </q> </p>

5- <cite> - Esa tag define a fonte de uma citação como nome do livro,autor, musica.
Exemplo do uso da tag:<p> No livro "A viuvinha", <cite> escritor José de Alencar </cite> </p>

- 6<address> Indica informações de contato (autor, organização)
Exemplo do uso da tag:<p> Vamos definir o contato de <address>

"Informações do escritor"
</address> </p>

7- <del> - Texto removido (ex: edição de conteúdo)
Exemplo do uso da tag:<p> <del> Esse texto foi Excluido </del> </p>

8- <ins> - Texto inserido (semântico para alterações em texto)
Exemplo do uso da tag:<p> Vamos Inseriri um texto<ins> Novo objeto inserido </ins> </p>

9- <sup> - indica que o conteúdo é um valor elevado (como em expoentes ou notas de rodapé).
Exemplo do uso da tag:<p> Vamos elevar um elemento do texto<sup> rodapé </sup> </p>

10- <sub> - indica que o conteúdo é um valor inferior, usado frequentemente em fórmulas químicas ou matemáticas.
Exemplo do uso da tag:<p> Vamos colocar um elemento do texto naparte inferior <sub> Elemento H20 </sub> </p>
